

# 账单介绍

## 账单概念

账单反映了用户每月在优刻得的消费金额以及资源购买详情，是优刻得与用户的对账凭据。

账单分为明细账单与汇总账单两种：

- **明细账单**：反映用户每月使用资源的扣费详情。
- **汇总账单**：将账单数据按照产品、项目、用户维度进行分别汇总，分为已支付汇总账单和未支付汇总账单。



## 账期

指订单入账的时间，例如，账期为2025年1月，体现的是订单的创建时间在2025年1月1日00:00-2025年2月1日00:00的交易账单。

对于历史未支付的订单，在支付完成后按支付时间进行入账。



### **【交易账单提醒】**

1. 交易账单描述您当月的资源购买情况；
2. 当日购买的资源将在次日零点前完成入账（注：后付费资源的订单创建时间如在次月，则在次月入账）；
3. **月账单的出账时间为每月1号6点整。e.g. 2022年1月的交易账单会在2022/02/01 6:00:00完成；**
4. **贵司对账单内容存在疑议，请在收到账单之日起10个工作日内告知我们。逾期未做任何回复的，我们将视为您认可账单内容。感谢您的理解与支持！**
